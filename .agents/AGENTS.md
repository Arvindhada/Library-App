
## Code Editing Strict Guidelines
- Hamesha pura code dhyan se check karo change karne ke baad.
- Aadha/Adhura code ya toote huye tags (jaise missing closing tags) mat chhodo.
- Kisi bhi replace ya edit function ke baad ek baar manually syntax check aur logic verify karo.
- User ko tabhi confirm karo jab tum 100% sure ho ki changes se syntax error nahi aayega.
